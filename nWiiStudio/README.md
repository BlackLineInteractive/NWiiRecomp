# nWiiStudio

## Overview
`nWiiStudio` is a Graphical User Interface (GUI) application designed for real-time debugging and state inspection of recompiled games.

## How it works
- **Interface:** Built using **Raylib** (for fast rendering) and **ImGui** (for interactive windows, buttons, and tables).
- **Monitoring:** Provides direct access to the CPU context, a Memory Viewer, hardware register values, and message queues.
- **Diagnostics:** Allows developers to track GX FIFO streams, HLE subsystem calls, and code execution to easily spot bugs and test emulation accuracy.
